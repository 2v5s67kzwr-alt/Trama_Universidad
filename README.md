# Trayectoria y Autonomía Neuroinclusiva 🌟

Presentación web interactiva desarrollada para **Trama Diversa**, enfocada en la exposición de programas de acompañamiento personalizados para la inclusión y autonomía de personas neurodivergentes en el entorno universitario.

## 🎨 Características del Proyecto
* **Formato 16:9 Adaptativo:** Diseñado con un contenedor rígido que se escala automáticamente para calzar perfecto en proyectores, tablets o computadores.
* **Navegación Interactiva:** Control de diapositivas mediante clics en botones en pantalla o utilizando las flechas del teclado y la barra espaciadora.
* **Identidad de Marca:** Estructurado bajo los colores corporativos oficiales de Trama Diversa.

## 🛠️ Tecnologías Utilizadas
* **HTML5 / CSS3** (Estructuras de animación de transiciones)
* **Tailwind CSS** (Estilos modernos y maquetación ágil)
* **JavaScript Nativo** (Lógica de control de diapositivas y auto-escalado)
* **FontAwesome** (Iconografía técnica y de soporte)
